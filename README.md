<h1>machine learning</h1>
<h2>1. what is maching learing?</h2>
<p>
    머신러닝(기계학습)이란 기계가 일일이 명시하지 않은 동작을 데이터로부터 스스로 학습하여 실행할 수 있도록 하는 알고리즘을 개발하는 연구 분야이다. 즉, 머신러닝을 통해 컴퓨터 시스템이 데이터를 학습할 수 있는 능력을 부여받게 된다. (여기서 <strong><u>학습</u></strong>이란 어떤 Task(T)에 대해서 이전의 Experience(E)를 이용해 Performance(P)를 향상시키는 것을 의미한다.)
</p>
<h3>1.1 applications</h3>
<p>
    머신 러닝은 이미 우리의 일상생활 속에서 공존하고 있다. 음성 인식, 스팸 메일 필터링, text 인식, 번역기, 챗봇, 얼굴 인식, sns 자동 친구 추천 등 머신 러닝을 이용해 제공되고 있는 서비스들이 다양하게 존재한다.
</p>
<h3>1.2 deep learning</h3>
<p>
    딥 러닝이란 머신러닝의 일부로, deep neural network 구조를 기반으로 한다. 이러한 딥 러닝 아키텍쳐는 컴퓨터 비젼, 음성 인식, 자연어 처리, 오디오 인식, 소셜 네트워크 필터링, 기계 번역, 생물 정보학, 약물 디자인을 포함해 다양한 분야에 적용되고 있다.
</p>
